# Python Formation

Coursework code from the [Alura's Formation in Python](https://cursos.alura.com.br/formacao-linguagem-python?preRequirementFrom=django).

This Formation includes the following courses:

1. Python: build your first application ([Website](https://cursos.alura.com.br/course/python-crie-sua-primeira-aplicacao)) - folder `sabor-express`;
2. Python: applying Object-Oriented Principles ([Webiste](https://cursos.alura.com.br/course/python-aplicando-orientacao-objetos));
3. Python: advance in Object-Oriented Principles and consume an API ([Website](https://cursos.alura.com.br/course/python-avance-orientacao-objetos-consuma-api)).
